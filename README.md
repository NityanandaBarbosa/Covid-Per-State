
# Covid-Per-State

## How to execute the App

###  Android/iOS
- You must conect your device in your computer.
- Inside flutter project dir run 
- $ flutter pub get
- $ flutter run

###  Chrome
- Inside flutter project dir run 
- $ flutter pub get
- $ flutter run -d Chrome

###  Edge
- Inside flutter project dir run 
- $ flutter pub get
- $ flutter run -d Edge
